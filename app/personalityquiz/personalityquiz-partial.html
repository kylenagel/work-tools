CMGO Games: Personality Quiz

Link: CMGO Games
What it does: User answers a series of seemingly unrelated questions, and the responses lead to an end result. User doesn't see result until after all the questions are answered.

How to make one

1. Before starting, confirm or set up a Google account

  • This is important because, not only are Google accounts very useful for creating and storing all kinds of Google products and information, but also because a Google Spreadsheet is needed to create the timeline.
  • Go to drive.google.com.
  • If you already have a Google account (a gmail address), sign in. If not, create one. It's free and easy. Just follow the instructions.

2. Create a blank Google Spreadsheet

  • Open your drive, drive.google.com.
  • Click the red "CREATE" button on the left and then "Spreadsheet."
  • Rename the spreadsheet to something that will be easy to find later. File > Rename.

3. Create two tabs in your spreadsheet

  • At the bottom of the spreadsheet, click the "+" sign to the left of the tab titled "Sheet1" to create a second tab

4. Set up the column names in each tab

  • First tab (questions and options)
    1. Question
    2. A column for each option, named "Option1," "Option2" and so on.
  • Second tab (results)
    1. Result (name of the result)
    2. Explanation
    3. Facebook Explanation
    4. Image (url for an image matching the result)
    5. Caption

5. Enter your content in each tab

  • First tab
    1. Question: the text of the question
    2. Option columns: the option the user sees with the result(s) between curly brackets.
      Text or image: You can input either text or image URLs for the options. If it's an image URL, the user will see an image for the options (URL must end in .jpg or .png). If it's text, the user will see the text.
      Multiple results tied to one option: One option can connect to multiple results by separating those results with ";" and no space in between.
      Example (text option): "Cookie Monster {Dayton Art Institute}" in a cell will show the user an option of "Cookie Monster" for that question. If the user chooses Cookie Monster, it will add 1 to the total for result "Dayton Art Institue."
      Example (image option): "http://media.cmgdigital.com/shared/img/photos/2015/04/02/8f/d2/040515-ham-main-street4.jpg {Bacon Fest}" in a cell will show the user the image from that URL as an option for that question. If the user chooses that image, it will add 1 to the total for the result "Bacon Fest."
      Example (multiple results with an option): "Ice Ice Baby {Century Bar;Blind Bob's}" in a cell will show the user an option of "Ice Ice Baby" for that question. If the user chooses Ice Ice Baby, it will add 1 to the total for both results "Century Bar" and "Blind Bob's"
      Important: The text for the matching result inside the curly brackets must exactly match the result text in the first column of the second tab. Otherwise, the quiz won't find a matching result whose total to add to.
  • Second tab
    1. Result: The name of the result. The user will later see the text after "You got ..." This text must also exactly match what's inside the curly brackets in the options fields.
    2. Explanation: The explanation of the result that shows to users when they complete the quiz.
    3. Facebook Explanation: The explanation of the result that shows on Facebook when users share their result (might need to be shorter or different than what is shown to users when they receive their result).
    4. Image: The image the user will see along with the result
    5. Caption: Caption to show below the result image

6. Publish your spreadsheet

  • File > Publish to the Web > Publish (yes, when asked, you're sure you want to publish)

7. Communicate the Google Spreadsheet URL to the digital team

  • The digital team can then create the quiz on a page on our websites
