CMGO Games: Knowledge Quiz

Link: CMGO Games
What it does: User sees a series of questions with multiple choice options. When clicking on an option, the user immediately sees whether it was right or wrong and gets an explanation.

How to make one

1. Before starting, confirm or set up a Google account

  • This is important because, not only are Google accounts very useful for creating and storing all kinds of Google products and information, but also because a Google Spreadsheet is needed to create the timeline.
  • Go to drive.google.com.
  • If you already have a Google account (a gmail address), sign in. If not, create one. It's free and easy. Just follow the instructions.

2. Create a blank Google Spreadsheet

  • Open your drive, drive.google.com.
  • Click the red "CREATE" button on the left and then "Spreadsheet."
  • Rename the spreadsheet to something that will be easy to find later. File > Rename.

3. Create two tabs in your spreadsheet

  • At the bottom of the spreadsheet, click the "+" sign to the left of the tab titled "Sheet1" to create a second tab

4. Set up the column names in each tab

  • First tab (questions and options)
    1. Question
    2. Answer
    3. A column for each option, named "Option1," "Option2" and so on.
    4. Explanation
    5. Photo
    6. Caption
  • Second tab (results)
    1. Correct_Answers_Min
    2. Correct_Answers_Max
    3. Message
    4. Image
    5. Caption

5. Enter your content in each tab

  • First tab
    1. Question: the text of the question
    2. Option columns: the options the user sees
      Text or image: You can input either text or image URLs for the options. If it's an image URL, the user will see an image for the option (URL must end in .JPG, .jpg or .png) and you must include the text of the option inside curly brackets after the URL. If it's text, the user will see the text.
      Example (image option): "http://media.cmgdigital.com/shared/img/photos/2015/04/01/39/7d/040515lifenobodiesdeyo.jpg {Orville Wright}" in a cell will show the user the image from that URL as an option for that question. If the user chooses that image, it will register as choosing the option "Orville Wright" (Note: As a best practice, don't include an image in the questions "Photo" cell if there are images for options.)
      Important: At least one option must exactly match the result text for that question
    3. Explanation: content the user will see after answering the question and being alerted whether the answer is correct or incorrect.
    4. Photo: a photo that can shows about the options (Note: Horizontal or square photos will look best.)
    5. Caption: a caption for the photo
  • Second tab
    1. Result_Answers_Min: Minimum number of correct answers for the user to see this message (Note: Be sure the first row in this tab has "0" for the minimum.)
    2. Result_Answers_Max: Maximum number of correct answers for the user to see this message (Note: As a best practice, make the final row max "100" in case the number of questions changes)
    3. Message: Content to display is the number of correct answers falls within the range signified in the first two columns
    4. Image: Image to display below the result message
    5. Caption: Caption for the image

6. Publish your spreadsheet

  • File > Publish to the Web > Publish (yes, when asked, you're sure you want to publish)

7. Communicate the Google Spreadsheet URL to the digital team

  • The digital team can then create the quiz on a page on our websites
