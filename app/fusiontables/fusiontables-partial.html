Google Fusion Tables

Link: Google Fusion Tables
What it does: Google built Fusion Tables as a tool to upload, manage and visualize data. It's best use is for mapping multiple (like, thousands) of addresses or creating maps of polygons (like states or counties) with a built-in information window for clicking.

How to make one

1. Before starting, confirm or set up a Google account

  • This is important because, not only are Google accounts very useful for creating and storing all kinds of Google products and information, but also because Google Fusion Tables files are automatically saved to your drive.
  • Go to drive.google.com.
  • If you already have a Google account (a gmail address), sign in. If not, create one. It's free and easy. Just follow the instructions.

2. Organize your data

  • It's easiest to work with your data in an Excel file before importing that file into Fusion Tables.
  • If you're making a map with markers, make sure one column includes the full addresses for locations where you want the markers to appear. Later, you'll be able to tell Fusion Tables which column contains the location information.
  • If you're making a map of polygons, make sure one of your columns includes information that later be used to merge yourr table with the boundaries information table. For instance, if the boundaries information table labels counties like "OH-Montgomery" and "OH-Warren," use the same naming convention in your column.

3. Import data to Fusion Tables

  • Open your drive, drive.google.com.
  • Click the red "CREATE" button on the left and then "Fusion Table."
  • Choose the Excel file you want to upload, then click "Next."
  • If your data looks correct in the second window, click "Next." If not, go back to your Excel file and fix any problems before going through the importing process again.
  • A third window allows you to enter information about your table, including a name (otherwise inherited from the Excel file), information about the data and a description.
  • Click "Finish" to import
If making a map of markers ...

4. Identify your column for locations

  • Hover over the column that holds your locations.
  • Click the down arrow next to the column name, then Change.
  • For Type, mark as Location, then click Save changes.
  • The text in the location column should now be colored yellow.

5. Make the map

  • Click the box with the red background and white + sign in the line of tabs and then Add Map.
  • Fusion Tables will geocode the column of addresses you've marker as locations. It will turn those addresses into latitude and longitude pairs, then place them on the map.

6. Adjust the markers

  • Tools > Change Map
  • In the left column, click "Change feature styles ..."
  • For markers, you have 10 choices by default: Red, yellow, green, blue and purple in dots and pins.
  • In more advanced cases, when you want to show different markers for different sets of values:
    • Inside the Change map feature styles box, click the Buckets tab.
    • Choose how many "buckets" - categories of markers - you'd like to have with the dropdown menu between "Divide into" and "buckets."
    • Choose the column whose values will serve as the values to determine which markers will display.
    • Pick the values whose rows will display as which markers. The values you type into the boxes are the lower ends of the range.
    • Pick the markers to display for those values in the dropdown menus next to the values.
If making a map of polygons ...

4. Merge your table with a table containing boundary information

  • Find the Fusion Table containing the boundary information for the areas you want to display. Some common boundary files, including states, counties and congressional districts, can be found here.
  • In your imported Fusion Table with your new data, go to File > Merge.
  • Paste the url of the boundary information Fusion Table into the box at the bottom, under "Or paste a web address here:" and click Next
  • From the dropdown menus, choose the columns from the tables that match.
  • Fusion Tables will make a new table combining the two. Your data for the desired areas will now have boundary information in the same row.

5. Make the map

  • Click the box with the red background and white + sign in the line of tabs and then Add Map.
  • Fusion Tables will read the boundary information in each row and create those polygons on the map.

6. Adjust the polygons

  • Tools > Change Map
  • In the left column, click "Change feature styles ..."
  • There are three things you can change about polygons: Fill color, Border color and Border width.
  • For color, you can choose from the available squares or enter a hex code, a six-digit alphanumeric color code that follows a # sign. An easy way to play with hex codes is with the site ColorPicker. Find the color you like, the paste the six digits after the # sign.
  • In more advanced cases, when you want to show different colors or borders for the polygons for different sets of values:
    • Inside the Change map feature styles box, click the Buckets tab.
    • Choose how many "buckets" - or categories - you'd like to have with the dropdown menu between "Divide into" and "buckets."
    • Choose the column whose values will serve as the values to determine which colors or features will display.
    • Pick the values whose rows will display as which colors or features. The values you type into the boxes are the lower ends of the range for thhat category.
    • Pick the colors or values to display for those values in the dropdown menus next to the values.

7. Adjust the information window

  • Click "Change info window..." in the left column.
  • On the left, click the boxes of the columns whose information you want to display in the information window when a user clicks on a marker or polygon.
  • By default, the values will show as the bolded column name and then the value for the same row as the marker or polygon the user clicked.

8. Publish the map to make it available to embed

  • Tools > Publish
  • At the top of the box, click "Change visibility"
  • On the line now listed as Private, click Change and choose "Public on the web."
  • Click save

9. Get the embed code

  • Tools > Publish
  • The code in the box underneath "Paste HTML to embed in a website" is what you'll need to embed on a website.
